What is Session Wizard?

Session Wizard is a clinical toolkit for planning, tracking, and monitoring clients' progress in speech-language pathology and cognitive rehabilitation. It was created by Lucas James, SLP-Reg, and is currently in active development. For full documentation, and to ask questions or report any issues, please visit the Help menu and access the online documentation.
