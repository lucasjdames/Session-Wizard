Speech Deck: Motor Speech Practice Stimuli

How to Use

Modes: Table for row-based scoring and comments; Flash-Card for focused, one-item practice (type a completion then score).

Scoring: Use + (Accurate), +/- (Partial), and - (Off-Target/NR) buttons.

Custom Decks: Paste a list (one per line) and click Create Custom Deck, or add single items in Table View. Custom decks persist when you close the pop-up.

Save: Always click Save Results before closing the popup to preserve your session data.

Shortcuts: ← / → (navigate), Esc (close).
