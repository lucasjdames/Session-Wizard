Session Wizard

Version 1.2.1

Session Wizard is a clinical toolkit for planning, tracking, and monitoring clients' progress in speech-language pathology and cognitive rehabilitation. It aims to facilitate goalsetting and data collection inside and outside of the therapy room. It was created by Lucas James, SLP-Reg. Please note that the project was developed with the assistance of large language models for coding and image generation. Session Wizard will always be free to use and open source.

The current tools available are:

Tools

  • Goal Builder — create SMART goals and build Goal Attainment Scales (GAS). Drag-and-drop components and checklists help ensure completeness.
  • Therapy Session Data Taker — customizable templates for real-time session tracking. Includes special components for common interventions (PACE, discourse transcription, spaced retrieval) and a flexible data table for trial-based scoring.
  • Homework Tracker — create printable home program logs with configurable activities, frequencies, and scheduling.

For full documentation and source code, visit GitHub — Session Wizard. Development is ongoing.
